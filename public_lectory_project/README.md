# Публичный лекторий — веб-проект

Веб-приложение создано на основе ТЗ для проекта «Публичный лекторий».

## Стек
- Python
- Flask
- SQLite3
- HTML
- CSS

## Возможности
- главная страница проекта;
- список лекций;
- список лекторов;
- регистрация слушателей;
- сбор обратной связи;
- простая административная панель для просмотра заявок и отзывов.

## Запуск
```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
pip install -r requirements.txt
python app.py
```

После запуска откройте в браузере:
```text
http://127.0.0.1:5000
```
